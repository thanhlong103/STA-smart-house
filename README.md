## Pin Configuration

### RFID Module
| Function | GPIO Pin |
|----------|---------|
| SDA (SS) | GPIO 5  |
| SCK      | GPIO 18 |
| MOSI     | GPIO 23 |
| MISO     | GPIO 19 |
| RST      | GPIO 22 |

### DHT11 Sensor
| Function | GPIO Pin |
|----------|---------|
| OUT      | GPIO 4  |

### Light Sensor
| Function | GPIO Pin |
|----------|---------|
| OUT      | GPIO 25 |

### NeoPixel LED
| Function | GPIO Pin |
|----------|---------|
| OUT      | GPIO 16 |
